授权信息模块是专属集成平台的核心许可管理中心，提供系统各功能模块的授权状态、有效期和使用权限的集中展示。通过该模块，系统管理员可直观了解当前租户的授权范围、各功能模块的授权期限，并及时处理授权即将到期的模块，确保系统功能的连续可用性。

## 功能说明

### 授权用户

显示开通租户时设置的授权用户名称，该信息在租户创建时由系统管理员或服务提供商配置，用于标识当前租户的归属。

### 授权模块与授权时间

系统以表格形式展示各功能模块的授权状态和有效期限：

| 授权模块 | 授权时间 | 状态 |
|----------|----------|------|
| 用户连接 | 2023-07-12 至 2026-02-01 | 正常 |
| 应用 SSO | 2023-07-12 至 2026-12-06 | 正常 |
| 安全基线监测/设备准入 | 2025-04-21 至 2026-03-30 | 正常 |
| 应用分发 | 2023-07-12 至 2026-01-31 | 正常 |
| 数据防泄漏 | 2023-07-12 至 2026-12-05 | 正常 |
| 安全接入网关 | 2025-02-11 至 2026-03-30 | 正常 |
| 数据连接 | 2024-02-01 至 2026-03-31 | 正常 |

<Callout type="warn">
模块到期后，相关功能将在管理页面中无法进行操作和配置。如需继续使用，请联系技术支持进行授权续期。
</Callout>

### 授权状态说明

| 状态 | 说明 |
|------|------|
| **正常** | 授权有效期内，功能可正常使用 |
| **即将到期** | 授权剩余时间少于 30 天，功能仍可使用，系统会显示提醒 |
| **已过期** | 授权时间已过期，相关功能无法在管理页面进行配置 |

### 授权模块功能说明

**用户连接**：提供身份源集成、用户管理、组织架构管理等核心能力，包含通讯录集成、通讯录同步等功能，是构建企业身份基础设施的基础模块。

**应用 SSO**：提供企业应用的单点登录、身份联合认证、应用门户等功能，支持 SAML、OIDC、CAS 等标准协议。

**安全基线监测/设备准入**：提供终端设备安全状态评估、设备合规性检测、设备安全基线管理等功能。

**应用分发**：支持企业移动应用的统一管理和分发，包括 iOS、Android 应用的版本控制、更新推送和权限管理。

**数据防泄漏**：提供文档水印、截屏控制、复制限制等 DLP 功能，防止敏感信息通过移动设备和应用泄露。

**安全接入网关**：提供零信任网络访问控制，支持细粒度的应用访问策略管理、流量加密和安全审计。

**数据连接**：提供企业系统间的数据集成和工作流自动化能力，支持多种协议和数据格式的转换。

### 申请授权续期

<Steps>
<Step>
### 步骤 1：联系技术支持团队

{/* DOC_IMAGE src="/images/参数配置/授权信息/step-04.png" alt="联系技术支持团队截图" width="1200" height="720" */}

当检测到模块授权即将到期或已过期时，联系专属集成平台技术支持团队。
</Step>
<Step>
### 步骤 2：提供租户信息

{/* DOC_IMAGE src="/images/参数配置/授权信息/step-05.png" alt="提供租户信息截图" width="1200" height="720" */}

提供租户 ID 和授权用户信息，并明确需要续期的功能模块。
</Step>
<Step>
### 步骤 3：完成续期手续

{/* DOC_IMAGE src="/images/参数配置/授权信息/step-06.png" alt="完成续期手续截图" width="1200" height="720" */}

签署续期协议并完成续期费用支付。
</Step>
<Step>
### 步骤 4：确认续期生效

{/* DOC_IMAGE src="/images/参数配置/授权信息/step-07.png" alt="确认续期生效截图" width="1200" height="720" */}

技术支持团队完成授权更新后，授权信息页面将自动刷新显示新的有效期，无需手动操作。
</Step>
</Steps>

<Callout type="info">
建议系统管理员每季度检查一次授权信息页面，提前规划授权续期工作，避免因授权过期导致业务中断。
</Callout>

## 常见问题

<Accordions>
<Accordion title="授权到期后，系统中已有的数据和配置会丢失吗？">
不会。授权到期仅会限制在管理界面进行新的配置更改，不会导致已有数据和配置丢失。但长期未续期可能导致相关功能服务暂停，建议及时续期。
</Accordion>
<Accordion title="如何确认授权续期是否生效？">
授权续期完成后，可在授权信息页面查看更新后的授权时间。系统会自动刷新授权信息，无需手动操作。
</Accordion>
<Accordion title="是否可以单独为某个模块申请授权续期？">
是的。专属集成平台支持针对单个功能模块进行授权续期，您可以根据业务需求选择需要续期的模块。
</Accordion>
<Accordion title="授权到期后已配置的功能还能正常运行吗？">
授权到期不会立即影响已配置的功能运行，但系统会定期进行授权检查，长期未续期可能导致相应功能暂停服务。
</Accordion>
</Accordions>
