## 概述

  src="/images/参数配置/品牌设置/企业消息配置/企业消息（钉钉／飞书）配置/overview.png"

本文档介绍如何在专属集成平台配置企业消息功能，支持通过钉钉和飞书发送消息通知。通过企业消息配置，您可以實現密码修改通知、密码重置通知、待办提醒等多种自动化消息推送，提升企业沟通效率。

## 前提条件

### 飞书前提条件

- 拥有飞书开发者后台的管理员权限
- 若无账号，请先前往[飞书开放平台](https://open.feishu.cn/)注册开发者账号
- 创建飞书应用，并开通消息相关权限
- 添加机器人能力

### 钉钉前提条件

- 拥有钉钉开发者后台的管理员权限
- 若无账号，请先前往[钉钉开放平台](https://open.dingtalk.com/)注册开发者账号
- 创建钉钉应用，并开通消息相关权限
- 添加机器人能力

- 配置步骤

### 步骤一：创建授权凭证

<ImageZoom
  src="/images/参数配置/品牌设置/企业消息配置/企业消息（钉钉／飞书）配置/step-01.png"
  alt="创建授权凭证截图"
  width={1200}
  height={720}
/>

- 进入**参数配置 > 授权管理**页面
1. 建立飞书或钉钉的授权连接
2. 输入已开通消息权限的应用 AppKey 和 AppSecret

| 平台 | 参数 | 说明 |
|------|------|------|
| 飞书 | AppKey | 飞书应用的应用标识 |
| 飞书 | AppSecret | 飞书应用的应用密钥 |
| 钉钉 | AppKey | 钉钉应用的应用标识 |
| 钉钉 | AppSecret | 钉钉应用的应用密钥 |

### 步骤二：开启企业消息服务

<ImageZoom
  src="/images/参数配置/品牌设置/企业消息配置/企业消息（钉钉／飞书）配置/step-02.png"
  alt="开启企业消息服务截图"
  width={1200}
  height={720}
/>

- 进入**品牌设置 > 企业消息配置**页面
1. 开启**企业消息服务配置**开关
- 在下拉列表中选择第一步配置的企业应用

### 步骤三：配置消息模板

<ImageZoom
  src="/images/参数配置/品牌设置/企业消息配置/企业消息（钉钉／飞书）配置/step-03.png"
  alt="配置消息模板截图"
  width={1200}
  height={720}
/>

- 切换到**消息模板**菜单
1. 开启需要使用的消息通知开关
2. 根据需要修改默认模板，实现定制消息推送

支持的消息类型：

| 消息类型 | 说明 | 默认开启 |
|----------|------|----------|
| 密码修改通知 | 用户修改密码后发送通知 | 否 |
| 密码重置通知 | 用户重置密码后发送通知 | 否 |
| 账号锁定通知 | 账号被锁定时发送通知 | 否 |
| 异地登录提醒 | 检测到异地登录时发送提醒 | 否 |
| 待办提醒 | 待办事项到期前发送提醒 | 否 |

### 步骤四：验证配置

<ImageZoom
  src="/images/参数配置/品牌设置/企业消息配置/企业消息（钉钉／飞书）配置/step-04.png"
  alt="验证配置截图"
  width={1200}
  height={720}
/>

配置完成后，进行以下验证：

- 执行密码修改操作，确认收到飞书/钉钉消息通知
1. 执行密码重置操作，确认收到消息通知
2. 如配置了其他消息类型，进行相应测试

## 消息模板语法

### 变量引用

消息模板支持使用变量，格式为 `{\{variable_name\}}`：

```
您好，{{user_name}}，您的账号于 {{time}} 进行了密码修改操作。
```

支持的系统变量：

| 变量名 | 说明 | 示例 |
|--------|------|------|
| user_name | 用户姓名 | 张三 |
| user_email | 用户邮箱 | zhangsan@company.com |
| time | 操作时间 | 2024-01-15 14:30:00 |
| app_name | 应用名称 | 专属集成平台 |
| ip_address | 操作 IP | 192.168.1.100 |

### 飞书消息格式

飞书支持富文本消息格式：

```json
{
  "msg_type": "text",
  "content": {
    "text": "消息内容，支持 {{variable}} 变量"
  }
}
```

### 钉钉消息格式

钉钉支持多种消息格式：

```json
{
  "msgtype": "text",
  "text": {
    "content": "消息内容，支持 {{variable}} 变量"
  }
}
```

## 常见问题

**Q1：消息发送失败如何排查？**

A1：可能原因及解决方案：
- 检查授权凭证（AppKey/AppSecret）是否正确
- 确认应用的消息权限是否已开通
- 检查机器人能力是否已添加
- 查看审计日志中的消息发送记录

**Q2：为什么收不到消息通知？**

A2：请检查以下配置：
- 确认企业消息服务开关已开启
- 确认对应的消息类型开关已开启
- 检查用户是否在消息接收范围内
- 确认用户的飞书/钉钉绑定信息正确

**Q3：如何自定义消息模板？**

A3：在消息模板配置页面，点击具体消息类型的"编辑"按钮，可以修改模板内容。支持添加变量、调整格式。注意保持变量格式正确，否则变量将显示为原始文本。

**Q4：支持哪些消息接收人范围？**

A4：消息可以发送给：
- 操作用户本人
- 用户所属部门负责人
- 指定管理角色
- 自定义用户列表

**Q5：消息有发送限制吗？**

A5：是的，各平台有相应的限制：
- 飞书：每个应用每分钟最多发送 100 条消息
- 钉钉：每个应用每分钟最多发送 50 条消息

超出限制时，系统会进行队列缓冲后重试发送。
