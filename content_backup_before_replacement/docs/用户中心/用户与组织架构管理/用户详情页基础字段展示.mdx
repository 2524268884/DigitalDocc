## 功能概述

专属集成平台的**用户详情页**现已支持展示和编辑全部平台内置基础字段，管理员可以在用户详情/编辑页中直接查看和修改所有系统标准字段，无需通过 API 操作。

**支持展示的基础字段（完整列表）：**

| 字段名称 | 字段类型 | 说明 |
|---------|---------|------|
| 账号 | string | 用户登录账号 |
| 姓名 | string | 用户真实姓名 |
| 手机号 | string | 手机号码 |
| 邮箱 | string | 邮箱地址 |
| 身份证号 | string | 居民身份证号码 |
| 昵称 | string | 用户昵称 |
| 性别 | enum | 男 / 女 / 未知 |
| 有效期 | date | 账号有效截止日期 |
| 工号 | string | 企业工号 |
| 地址 | string | 通讯地址 |
| 生日 | date | 出生日期 |
| 固定电话 | string | 办公室座机号码 |
| 个人主页 | string | 个人网站 URL |
| 头像 | image | 用户头像图片 |

## 查看用户基础字段

<Steps>
<Step>
### 步骤一：进入用户详情页

在专属集成平台管理控制台中，进入 **用户中心 > 用户管理**，点击目标用户进入用户详情页。
</Step>
<Step>
### 步骤二：查看基础字段信息

在用户详情页的「基础信息」选项卡中，可以查看所有已填写的基础字段值。未填写的字段显示为空。

<ImageZoom src="/images/user/user-detail-basic-fields.png" alt="用户详情页基础字段展示" width={800} height={450} />
</Step>
</Steps>

## 编辑用户基础字段

<Steps>
<Step>
### 步骤一：进入编辑模式

在用户详情页，点击右上角「编辑」按钮，进入编辑模式。
</Step>
<Step>
### 步骤二：修改字段值

在各字段输入框中修改对应的值：

<Callout type="info">
部分字段（如账号、创建时间、创建人）为系统只读字段，在编辑模式下不可修改。
</Callout>
</Step>
<Step>
### 步骤三：保存修改

点击「保存」完成修改。字段值将立即更新，并可通过同步连接器下发至下游系统。
</Step>
</Steps>

## 配置字段可见性

如需控制特定基础字段在用户详情页的展示与否，可在 **用户中心 > 属性管理** 中对基础字段进行可见性配置：

| 配置项 | 说明 |
|--------|------|
| 管理员可见 | 是否在管理控制台用户详情页展示此字段 |
| 用户自助可见 | 是否在用户自助门户的个人信息页展示此字段 |
| 用户自助可编辑 | 用户是否可以在自助门户自行修改此字段 |
