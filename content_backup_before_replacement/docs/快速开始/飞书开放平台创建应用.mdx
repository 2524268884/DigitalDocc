## 概述

在将飞书接入专属集成平台之前，需要先在飞书开放平台创建一个企业自建应用，开通通讯录相关权限并获取凭证。

## 前提条件

| # | 条件 | 验证方式 |
|---|------|---------|
| 1 | 飞书企业管理员账号 | 能登录[飞书开放平台](https://open.feishu.cn/) |
| 2 | 专属集成平台管理员账号 | 能登录集成平台管理控制台 |

---

## 飞书侧配置

<Steps>
<Step>
### 创建企业自建应用

1. 登录[飞书开放平台](https://open.feishu.cn/)，点击右上角「开发者后台」
2. 点击「创建应用」，选择**企业自建应用**
3. 填写以下信息并提交：

| 字段 | 说明 |
|------|------|
| 应用名称 | 填写便于识别的名称，如「专属集成平台」 |
| 应用图标 | 上传应用图标 |
| 应用描述 | 简要描述应用用途 |

</Step>
<Step>
### 获取 AppID 和 AppSecret

<ImageZoom
  src="/images/快速开始/飞书开放平台创建应用/overview.png"
  alt="创建企业自建应用截图"
  width={1200}
  height={720}
/>

1. 在应用详情左侧菜单选择**凭证与基础信息**
2. 记录以下参数：

| 参数 | 说明 |
|------|------|
| `App ID` | 应用唯一标识 |
| `App Secret` | 应用密钥 |

<Callout type="warn">
App Secret 是敏感信息，请妥善保管，不要泄露给无关人员。
</Callout>
</Step>
<Step>
### 开通通讯录权限

<ImageZoom
  src="/images/快速开始/飞书开放平台创建应用/step-03.png"
  alt="开通通讯录权限截图"
  width={1200}
  height={720}
/>

1. 在应用详情左侧菜单选择**权限管理**
2. 搜索并开通以下权限：

**用户身份权限**

| 权限名称 | 说明 |
|---------|------|
| 获取用户 ID | 获取用户唯一标识 |
| 获取用户基本信息 | 姓名、头像等基本信息 |
| 获取用户邮箱 | 用于账号关联 |
| 获取用户手机号 | 用于账号关联 |

**通讯录权限**

| 权限名称 | 说明 |
|---------|------|
| 获取通讯录组织架构信息 | 同步部门层级 |
| 获取通讯录单位信息 | 同步组织单元 |
| 获取通讯录部门基本信息 | 同步部门信息 |
| 获取通讯录部门组织架构信息 | 同步部门层级关系 |
| 获取通讯录用户基本信息 | 同步用户信息 |

<Callout type="warn">
通讯录相关权限的「数据范围」必须设置为**全部成员**，否则只能同步部分用户数据。
</Callout>
</Step>
<Step>
### 发布应用

<ImageZoom
  src="/images/快速开始/飞书开放平台创建应用/step-05.png"
  alt="发布应用截图"
  width={1200}
  height={720}
/>

1. 在左侧菜单选择**应用发布 > 版本管理与发布**
2. 点击「创建版本」，填写版本号（如 `1.0.0`）和更新说明
3. 将可用性状态设置为「所有员工」
4. 提交发布申请
</Step>
<Step>
### 企业管理员审核

<ImageZoom
  src="/images/快速开始/飞书开放平台创建应用/step-06.png"
  alt="企业管理员审核截图"
  width={1200}
  height={720}
/>

1. 登录飞书管理后台（非开放平台）
2. 进入**工作台 > 应用审核**
3. 找到待审核应用，确认权限和信息后点击「通过」

<Callout type="info">
应用发布需要企业管理员审核，请提前与管理员沟通，避免延误。
</Callout>
</Step>
</Steps>

## 集成平台侧配置

<Steps>
<Step>
### 添加飞书授权

<ImageZoom
  src="/images/快速开始/飞书开放平台创建应用/step-01.png"
  alt="添加飞书授权截图"
  width={1200}
  height={720}
/>

1. 登录专属集成平台管理控制台
2. 进入**系统设置 > 授权管理**
3. 点击「添加授权」，选择「飞书」
</Step>
<Step>
### 填写凭证信息

{/* DOC_IMAGE src="/images/快速开始/飞书开放平台创建应用/step-07.png" alt="填写凭证信息截图" width="1200" height="720" */}

| 配置项 | 说明 |
|--------|------|
| 授权名称 | 自定义名称，如「飞书生产环境」 |
| App ID | 飞书应用的 App ID |
| App Secret | 飞书应用的 App Secret |

填写完成后点击「确定」保存。
</Step>
</Steps>

---

## 验证配置

点击授权记录旁的「测试连接」，确认返回成功状态。

如连接失败，请检查：
- App ID / App Secret 是否正确（注意复制时不含多余空格）
- 权限范围是否设置为「全部成员」
- 应用是否已通过管理员审核

---

## 常见问题

<Accordions>
<Accordion title="为什么只同步了部分用户？">
最常见原因是通讯录权限的**数据范围未设为「全部成员」**。请在飞书开放平台 → 权限管理中检查每个通讯录权限的数据范围设置。
</Accordion>
<Accordion title="应用审核一直未通过怎么办？">
检查以下几点：应用信息是否填写完整、申请的权限是否有充分的用途说明。如权限申请过多，建议只申请实际需要的权限，并在申请说明中注明用途。
</Accordion>
<Accordion title="App Secret 泄露了怎么办？">
立即在飞书开放平台 → 凭证与基础信息页面重置 App Secret，并同步更新集成平台的授权配置。旧的 App Secret 重置后立即失效。
</Accordion>
</Accordions>
