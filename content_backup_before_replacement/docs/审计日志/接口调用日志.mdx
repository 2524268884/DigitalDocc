## 概述

<ImageZoom
  src="/images/审计日志/接口调用日志/overview.png"
  alt="接口调用日志概览截图"
  width={1200}

接口调用日志模块用于记录和展示专属集成平台接口调用的详细情况，包括开放平台接口和通讯录集成时的调用记录。管理员可通过该模块快速查看上下游数据推送与同步状态，排查异常问题，支持筛选、排序和导出功能，提升运维效率。

<Callout type="info">
日志默认保存最近 **180 天**，具体保存时间可在系统设置中查看或修改。
</Callout>

## 界面说明

日志列表展示以下字段：

| 字段 | 说明 |
|-----|------|
| 调用时间 | 接口调用的时间 |
| 日志 ID | 日志的唯一标识 |
| 来源系统 | 调用来源的系统或集成名称 |
| 接口类型 | 调用的接口类型 |
| 耗时 | 接口调用耗时 |
| 状态码 | HTTP 状态码 |
| 成功/失败 | 调用结果状态 |

点击行左侧展开按钮，可查看请求报文和返回报文详情。

## 操作步骤

### 步骤 1：进入接口调用日志页面

{/* DOC_IMAGE src="/images/审计日志/接口调用日志/step-01.png" alt="进入接口调用日志页面截图" width="1200" height="720" */}

在左侧导航栏选择 **审计日志** → **接口调用日志**。

### 步骤 2：查看日志列表

{/* DOC_IMAGE src="/images/审计日志/接口调用日志/step-02.png" alt="查看日志列表截图" width="1200" height="720" */}

默认显示当日的调用记录，点击行左侧展开按钮可查看请求报文和返回报文详情。

### 步骤 3：筛选日志

{/* DOC_IMAGE src="/images/审计日志/接口调用日志/step-03.png" alt="筛选日志截图" width="1200" height="720" */}

1. 在页面顶部使用日期筛选器选择时间范围（默认今日）
2. 点击「高级筛选」按钮，弹出配置弹窗
3. 选择筛选条件：
   - **调用来源/同步系统**：全部或具体集成/同步名称（包括已删除的）
   - **调用状态**：全部、成功、失败
   - **接口类型**：用户新增/更新/删除、部门新增/更新/删除、角色新增/更新/删除
   - **请求内容搜索**：输入关键词匹配请求报文
   - **响应内容搜索**：输入关键词匹配响应报文
4. 点击「确定」应用筛选；点击「重置」清除条件

<Callout type="warn">
高级筛选按钮在设置条件后将高亮显示。筛选后列表更新，支持分页（默认 10 条/页）。
</Callout>

### 步骤 4：排序日志

{/* DOC_IMAGE src="/images/审计日志/接口调用日志/step-04.png" alt="排序日志截图" width="1200" height="720" */}

点击列表头部「接口调用时间」列，可进行升序/降序排序。

### 步骤 5：导出日志

{/* DOC_IMAGE src="/images/审计日志/接口调用日志/step-05.png" alt="导出日志截图" width="1200" height="720" */}

- 若无筛选条件，点击「导出全部数据」按钮
- 若有筛选条件，点击「导出筛选数据（总数）」按钮

系统将生成 Excel 文件下载，文件格式为 XLSX。

## 常见问题

<Accordions>
<Accordion title="支持按请求/响应内容搜索吗？">

支持关键词搜索，可通过「请求内容搜索」和「响应内容搜索」输入关键词匹配请求报文或响应报文。
</Accordion>
<Accordion title="导出的 Excel 文件包含哪些字段？">

导出文件包含日志 ID、时间、类型等字段。
</Accordion>
</Accordions>
