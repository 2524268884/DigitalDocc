## 1\. 概述

本文档详细介绍如何通过专属集成平台配置中国电子云邮箱的单点登录功能。通过此配置，企业用户可以使用统一的身份认证访问中国电子云邮箱，无需重复登录，提升用户体验和工作效率。

## 2\. 配置步骤

### 2.1 创建中国电子云邮箱应用
1. 登录专属集成平台管理控制台
2. 导航至"应用中心 → 应用管理 → 应用市场"
3. 在搜索框中输入"中国电子云邮箱"
4. 点击"添加"按钮，将该应用添加到应用列表中

### 2.2 基本信息配置

在添加应用后，进入应用配置页面，设置基本信息：

![image.png](/images/单点登录-中国电子云邮箱/02b00381-7934-4d65-a5a9-6c69c15a9d4a.png)

| 基本信息参数 | 说明 |
|------------------|------|
| 自定义应用名称 | 根据企业需求自定义应用名称 |
| 应用分类 | 选择合适的应用分类，也可以自定义新增分类 |
| 首页地址 | 个人Portal，访问应用的地址 |
| Client ID | 应用的唯一标识，系统自动生成，不可编辑 |
| Client Secret | 应用的密钥，系统自动生成，不可编辑，但可以点击"重置"按钮重新生成 |
> **注意**：请记录Client ID和Client Secret，后续配置中国电子云邮箱单点登录时需要使用。   

### 2.3 登录配置

切换到"登录配置"选项卡，配置以下内容：

![image.png](/images/单点登录-中国电子云邮箱/06bbde35-aff1-44a5-9cee-fb0053b2576d.png)

| 登录配置参数 | 说明 |
|------------------|------|
| 本地登录方式 | 控制访问该应用的登录方式，可选择"IAM用户名密码"、"短信验证码"、"邮箱验证码"等方式 |
| 企业认证登录 | 根据"企业认证源"中的认证方式进行登录。勾选视为选中，应用将支持该登录方式 |
| 多因子认证 | 通过企业认证源或本地方式认证通过后，会触发短信或邮箱的二次验证；不勾选视为不开启 |
| 访问地址 | 应用的登录地址，用作回调地址 |
| 单点登录地址 | 根据实际情况填写中国电子云邮箱信息 |
| partnerid | 合作伙伴ID，从中国电子云邮箱获取 |
| 员工所属企业id | 企业在中国电子云邮箱的唯一标识 |
| key | 安全密钥，从中国电子云邮箱获取 |
| 用户身份 | 用户标识字段，通常使用邮箱地址 |
> **重要提示**：partnerid、员工所属企业id和key等参数需要从中国电子云邮箱服务提供方获取，请确保信息准确无误。   

### 2.4 授权范围配置

配置可访问此应用的用户范围。专属集成平台支持以下三种授权方式：
1. **基于组织架构和用户授权**：
    - 可选择特定的部门或用户进行授权
    - 适合按组织结构管理访问权限的场景
    - [基于组织架构和用户授权](https://alidocs.dingtalk.com/api/doc/transit?dentryUuid=YMyQA2dXW7lx40bGtweyG4XPVzlwrZgb&queryString=utm_medium%3Ddingdoc_doc_plugin_card%26utm_source%3Ddingdoc_doc)

2. **基于角色授权(RBAC)**：
    - 通过静态标签(角色)进行授权
    - 适合按职能角色管理访问权限的场景
    - [基于角色授权（RBAC）](https://alidocs.dingtalk.com/api/doc/transit?dentryUuid=YMyQA2dXW7lx40bGtweyXYRvVzlwrZgb&queryString=utm_medium%3Ddingdoc_doc_plugin_card%26utm_source%3Ddingdoc_doc)

3. **基于用户属性授权(ABAC)**：
    - 通过动态标签(用户属性)进行授权
    - 适合按用户特定属性管理访问权限的场景
    - [基于用户属性授权（ABAC）](https://alidocs.dingtalk.com/api/doc/transit?dentryUuid=Obva6QBXJwmEvXDPHzabB7ngWn4qY5Pr&queryString=utm_medium%3Ddingdoc_doc_plugin_card%26utm_source%3Ddingdoc_doc)


选择适合贵企业的授权方式，并配置相应的授权范围。

## 3\. 中国电子云邮箱端配置
> **说明**：本部分需要中国电子云邮箱管理员进行配置，或联系中国电子云邮箱技术支持协助完成。   

中国电子云邮箱端的配置通常包括以下步骤：
1. 登录中国电子云邮箱管理后台
2. 导航至单点登录配置页面
3. 填写专属集成平台提供的Client ID和其他必要信息
4. 配置安全设置和用户映射关系
5. 启用单点登录功能

具体配置步骤可能因中国电子云邮箱版本和企业实施方案而异，建议联系中国电子云邮箱技术支持获取详细指导。

## 4\. 验证单点登录

配置完成后，可通过以下步骤验证单点登录功能：
1. 登录专属集成平台用户门户
2. 在应用列表中找到并点击"中国电子云邮箱"应用图标
3. 系统应自动完成认证并跳转至中国电子云邮箱界面
4. 验证是否成功进入邮箱且无需再次输入账号密码

## 5\. 常见问题与排障

### 5.1 单点登录失败

**可能原因**：
- 配置参数错误，如partnerid、企业id或key不正确
- 用户身份映射不匹配
- 网络连接问题

**解决方案**：
- 检查并更正配置参数
- 确认用户在两个系统中的账号映射关系
- 验证网络连通性

### 5.2 登录后账号不匹配

**可能原因**：
- 用户身份字段配置不当
- 中国电子云邮箱的用户账号与专属集成平台的用户账号不一致

**解决方案**：
- 检查用户身份字段配置
- 确保两个系统中的用户信息保持一致
- 必要时进行用户数据同步

### 5.3 访问权限问题

**可能原因**：
- 用户未在专属集成平台的授权范围内
- 中国电子云邮箱中的用户权限设置有误

**解决方案**：
- 检查授权范围配置，确保用户已获得授权
- 验证中国电子云邮箱中的用户权限设置

## 6\. 最佳实践

### 6.1 安全建议
- 定期更新Client Secret，提高系统安全性
- 使用多因子认证增强身份验证强度
- 严格控制授权范围，实施最小权限原则
- 定期审查访问日志，监控异常登录行为
