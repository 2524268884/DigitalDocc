> 本指南帮助您在 5 分钟内完成最小可行的集成配置，体验专属集成平台的核心功能。

## 概述

本文档介绍最简化的专属集成平台接入路径，适合首次体验或快速验证概念的场景。通过本指南，您将完成：
- 创建第一个应用
- 配置单点登录（SSO）
- 实现基本的用户同步

**预计时间**：5 分钟

## 前置条件

- 已开通专属集成平台账号
- 拥有管理员权限
- 准备一个用于测试的第三方应用（如钉钉、飞书或企业微信）

## 步骤一：创建应用

1. 登录专属集成平台，进入 **应用中心 > 应用管理**
2. 点击 **「添加应用」**
3. 选择应用类型（建议选择「自定义应用」）
4. 填写应用基本信息：
   - 应用名称：如「测试应用」
   - 应用描述：可选
5. 点击 **「保存」**

## 步骤二：配置单点登录

1. 在应用详情页，点击 **「单点配置」** 标签
2. 启用 **「单点登录」** 开关
3. 选择认证协议（建议选择 OIDC，适用于大多数场景）
4. 配置回调地址：
   - 如果是自定义应用，填写您的应用回调地址
   - 如果是钉钉/飞书/企业微信应用，使用平台提供的标准回调地址
5. 点击 **「保存」**

## 步骤三：创建第一个身份源（可选）

如果您有 AD、钉钉、飞书等身份源，可以快速配置：

### 以钉钉为例

1. 进入 **用户中心 > 认证源管理**
2. 点击 **「新建身份源」**
3. 选择「企业认证源」> 「钉钉」
4. 填写钉钉应用的 AppKey 和 AppSecret
5. 点击 **「保存」**

## 步骤四：测试验证

1. 打开一个新的浏览器窗口（或使用隐身模式）
2. 访问您的应用登录页面
3. 点击「使用 xxx 登录」按钮
4. 使用测试账号完成登录
5. 验证是否成功登录并跳转到应用首页

## 常见问题

### Q: 登录失败怎么排查？

**A**: 检查以下几项：
1. 回调地址是否正确配置
2. 应用是否已授权给当前用户
3. 身份源配置是否正确
4. 查看审计日志中的错误信息

### Q: 回调地址是什么？

**A**: 回调地址是您的应用收到认证成功通知的 URL。在专属集成平台配置时，平台会自动生成一个回调地址，您需要将该地址配置到您的应用中。

### Q: 需要准备什么证书吗？

**A**: 对于测试环境，平台会提供默认证书。生产环境建议配置正式的 SSL 证书。
