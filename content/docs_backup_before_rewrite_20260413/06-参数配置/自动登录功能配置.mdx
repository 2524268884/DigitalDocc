## 概述

自动登录功能基于现有的"记住密码"功能，进一步优化登录体验。用户可勾选"自动登录"选项，系统将在检测到有效凭证时自动完成登录，提升高频用户的便捷性。该功能仅适用于 IDAAS 账密登录，并支持后台统一开关控制。



## 详细操作

### 管理员配置自动登录开关

**目的**：管理员通过后台控制是否启用自动登录功能，确保安全性和合规性。

![image.png](/images/自动登录功能配置/be93159c-4dd0-43ea-9b6a-f436352e9586.png)
1. 登录后台管理系统，进入【平台设置】-【密码策略】页面。
2. 确保【登录页是否开启记住密码】选项已开启。
3. 勾选【是否允许自动登录】选项。
4. 点击【保存】按钮，配置生效。

**注意事项**：自动登录开关依赖记住密码功能；本地凭证有效时间遵循记住密码逻辑；启用后，所有用户均可选择自动登录。

### 用户启用自动登录

**目的**：用户在登录时勾选自动登录，下次访问时无需手动操作即可自动登录。

![登录页面自动登录选项](https://alidocs.oss-cn-zhangjiakou.aliyuncs.com/res/mPdnpEbxJPo8gqw9/img/ee91a729-dbe7-4218-82a1-cdaab5366d4e.png)
1. 打开 IDAAS 登录页面，输入用户名和密码。
2. 勾选【自动登录】复选框（仅 IDAAS 账密登录时显示）。
3. 点击【登录】按钮，完成登录。
4. 下次访问登录页面时，系统将自动检测凭证并登录（若启用 MFA，仍需完成认证）。

**注意事项**：自动登录仅适用于 IDAAS 账密登录；若凭证过期或验证失败，系统将清除本地凭证，用户需重新登录；手动登出时将清除自动登录凭证。

### 与 MFA 功能的适配

**目的**：确保自动登录与多因素认证（MFA）兼容，维护安全性。
1. 管理员启用自动登录后，若租户同时启用 MFA，自动登录过程将触发 MFA 认证。
2. 用户勾选自动登录并成功账密登录后，系统跳转至 MFA 验证页面。
3. 完成 MFA 认证（如输入验证码）后，方可进入系统。

**注意事项**：MFA 认证不能跳过，即使自动登录成功；若 MFA 验证失败，用户需重新手动登录。
