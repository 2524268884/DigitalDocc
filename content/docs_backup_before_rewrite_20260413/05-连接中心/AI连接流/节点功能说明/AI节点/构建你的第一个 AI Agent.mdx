本篇文档将带你快速掌握如何在 **AI 连接流** 中配置一个 **AI Agent 节点**，并实现一个具备基础对话能力的智能输出流程。

你将学会：
- 在流程中插入 AI Agent 节点
- 配置所需的模型、内存与工具子节点
- 进行节点测试，查看 AI 响应效果

通过本教程，你可以构建一个可自动理解输入并输出自然语言回复的 AI Agent，为更复杂的智能业务流程打下基础。

### 🥇 步骤一：插入 AI Agent 节点
1. 进入 AI 连接流配置页面。
2. 点击画布上的「\+」，选择“AI Agent 节点”。
3. 拖拽到合适位置，并连接上游节点（如触发节点）。
4. 在 AI Agent 节点中配置 Agent 的系统消息(system)  以及用户的提问消息(prompt)



### 🧠 步骤二：配置模型节点（语言理解核心）
1. 点击 AI Agent 节点下方的 model 插槽
2. 在右侧的抽屉中选择你需要使用的模型服务
3. 配置模型的授权信息、并选择指定模型
4. 配置模型的详细参数设定（如温度、最大响应长度等）。



### 🧠 步骤三：配置内存节点（上下文能力）
1. 点击 AI Agent 节点下方的 memory 插槽。
2. 选择内存存储方式。
3. 完成模型参数配置：
    - **消息 key**：用于标识当前对话的唯一标识符，AI 会根据该 key 查询所有相关历史消息，实现上下文保持。
    - **上下文窗口长度**：设定 AI 在响应时最多引用的历史消息条数。



### 🧪 步骤四：测试对话效果
1. 重新打开 AI Agent 「节点配置区域」。
2. 点击测试，查看 AI Agent 返回的内容。
3. 如需调整效果，可修改提示词或模型参数。
