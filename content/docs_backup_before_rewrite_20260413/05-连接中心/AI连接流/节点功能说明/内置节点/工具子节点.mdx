### **🧩 工具节点的作用**

调用外部 API 或系统服务，实现指令执行

结合模型意图识别，实现智能工具调用

支持参数配置、响应处理与返回信息回写

实现“感知—理解—执行”闭环中的“执行”部分

### **🛠️ 添加与配置工具节点**
1. 插入工具节点

在 AI Agent 节点下方点击「tools 插槽」

选择“工具节点”，插入默认模板
2. 选择工具类型（当前版本仅支持 MCP 工具）

### **🔄 工具节点调用逻辑**

模型节点识别到特定意图（如“我要报销”）

AI 自动选择匹配的工具节点（如报销系统 API）

调用成功后结果返回，交由模型生成最终回答或传给下游节点
