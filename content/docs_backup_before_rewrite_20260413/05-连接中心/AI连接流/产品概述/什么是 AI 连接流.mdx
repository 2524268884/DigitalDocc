> AI 连接流是新一代智能专属集成平台，通过 AI 赋能的 iPaaS（专属集成平台即服务），为企业提供跨系统智能协同解决方案。   



:::
**智能数据理解与转换**

通过 AI 理解不同系统的数据，自动完成对应的数据映射
:::

:::
**上下文感知决策**

根据历史数据和当前流程逻辑进行智能判断，完成结果判断、条件流转等智能调度操作
:::

:::
**非结构化内容处理**

通过多模态模型能力，自动提取图片、pdf 等文件中的关键信息并进行结构化输出
:::

:::
**智能网页交互**

由大模型理解网页内容，并不断使用网页的结构变化，自动完成复杂网页操作
:::



# **AI 连接流的优势**

**多样化数据输入渠道**相比于传统 AI 产品的 chatbox 输入模式，AgenticFlow 允许使用第三方系统数据作为大模型的 prompt 使用
**AI 节点支持**提供多样化的 AI 节点能力，如 Agent节点、意图识别节点等，同时提供内置的 RAG 能力，允许企业结合自身特性完成自定义配置

**预置的第三方系统支持**集成国内 800\+ 系统应用，简单配置即可完成系统之间的数据对接，同时支持通过 MCP SERVER 能力由 AI 进行智能调度处理
**AI \+ Browser 能力支持**通过大模型识别用户诉求，智能模仿人工行为进行网页操作，实现获取网页数据或完成数据录入等业务高频场景
