title: "登录页CSS自定义页面样式"
description: "介绍如何在登录页面进行CSS自定义样式修改。"
---

如想对登录页面，进行CSS自定义样式修改，可在"品牌设置-登录页面设置"，"登录页设置"的表单内容中，开启"自定义页面样式"。在代码编辑器中，输入CSS样式，进行修改。

<img src="/images/登录页CSS自定义页面样式_images/b74fb5ff-7673-4d28-a62a-e6596f9948fb.png" alt="登录页设置" />

例如想要修改标题字体颜色，可在代码编辑器中输入

```css
.digitalsee-login-header-lead {
  color: blue;
}
```

左侧的预览效果，即可实时响应修改结果

<img src="/images/登录页CSS自定义页面样式_images/eede2174-d2b7-4636-82b6-a7ebb1ad7100.png" alt="预览效果" />
