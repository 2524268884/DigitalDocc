## 接口定位

- 所属能力分组：用户管理
- 典型用途：适合单账号核验、更新前校验和问题定位。
- 请求路径：`/iam/api/user`

## 调用建议

- 如果你已经知道用户帐号，这是比“查询用户”更直接的单条详情接口。
- 适合作为修改用户、启停用户和修改密码前的前置校验。
