## 接口定位

- 所属能力分组：用户管理
- 典型用途：建议作为用户联调的首条验证接口。
- 请求路径：`/iam/api/users`

## 调用建议

- 首次联调优先调用该接口，确认 Bearer Token 是否有效。
- 如果只需要获取单个账号详情，可改用“获取用户详情”。
- 如果需要复杂过滤条件，使用“过滤查询用户”会更直接。
