## 接口定位

- 所属能力分组：用户管理
- 典型用途：适合主数据同步、新用户开户和初始化导入。
- 请求路径：`/iam/api/users`

## 调用建议

- 创建前建议先调用“查询用户”确认账号是否已存在。
- 如果接口返回参数错误，优先核对必填字段和字段格式。
- 创建成功后，可通过“获取用户详情”或“查询用户”立即校验结果。
