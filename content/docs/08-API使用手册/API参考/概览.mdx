## 如何使用这一部分

- 如果你是第一次接入，请先阅读《快速开始》和《认证与请求规范》。
- 如果你已经知道要调用哪类能力，直接进入对应分组的概览页。
- 每个分组页都提供推荐调用顺序、高频接口和完整接口列表。

| 分组 | 接口数 | 典型用途 | 入口 |
| --- | --- | --- | --- |
| 认证鉴权 | 2 | 首次接入、联调前准备、Token 刷新策略确认。 | [查看分组](/docs/08-API使用手册/API参考/认证鉴权/概览) |
| 组织与部门 | 7 | 组织架构同步、部门初始化、按部门维度查询成员范围。 | [查看分组](/docs/08-API使用手册/API参考/组织与部门/概览) |
| 用户管理 | 8 | 账号同步、用户主数据维护、批量排查用户状态。 | [查看分组](/docs/08-API使用手册/API参考/用户管理/概览) |
| 角色与标签 | 10 | 角色初始化、按应用查询角色、为用户批量授予角色。 | [查看分组](/docs/08-API使用手册/API参考/角色与标签/概览) |
