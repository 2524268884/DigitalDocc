## 接口定位

- 所属能力分组：组织与部门
- 典型用途：用于组织初始化或从外部系统同步新部门。
- 请求路径：`/iam/api/orgs`

## 调用建议

- 创建前建议先调用“获取部门列表”或“过滤部门”，避免重复创建。
- 如果需要维护上下级关系，请先确认父部门标识在平台内已经存在。
- 创建成功后，可再调用“获取部门详情”做结果校验。
