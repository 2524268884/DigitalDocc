## 接口定位

- 所属能力分组：事件通知
- 典型用途：适合连接器回调和事件驱动的数据同步。
- 请求路径：`/iam/api/open/event/{tenantId}/{connectorId}/{apiKey}`

## 调用建议

- 调用前先确认 tenantId 与 connectorId 对应关系无误。
- 如果回调处理失败，优先排查 apiKey、事件体结构和连接器运行状态。
