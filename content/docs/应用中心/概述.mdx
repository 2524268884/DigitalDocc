## 什么是应用中心

应用中心是数犀集成平台的核心模块，负责统一管理企业中所有需要集成身份认证的应用。通过应用中心，您可以：

- **统一管控**：集中管理所有业务应用的访问入口
- **单点登录**：用户一次登录，访问所有已授权应用
- **安全授权**：精细化控制哪些用户可以访问哪些应用

## 核心功能

| 功能 | 说明 | 适用场景 |
|------|------|---------|
| [应用创建与配置](./应用创建与配置) | 创建并配置应用的基本信息 | 新增应用接入时 |
| [单点登录（SSO）](./单点登录配置) | 配置 OIDC/SAML/OAuth2/CAS 协议 | 应用免密登录 |
| [应用授权管理](./应用授权管理) | 控制用户/角色对应用的访问权限 | 权限精细化控制 |
| [应用密码代填](./应用密码代填) | 自动填充账号密码完成登录 | 不支持 SSO 的应用 |
| [应用网关](./应用网关) | 反向代理 + 身份验证 | 内网应用安全暴露 |
| [M2M 接口授权](./M2M接口授权/接口调用权限配置说明) | 服务间调用的字段级权限控制 | 下游系统 API 调用 |
| [单点登出](./单点登出配置) | 统一登出所有已登录应用 | 安全退出场景 |

## 支持的 SSO 协议

- **OIDC**（推荐）：现代标准，支持 Token 刷新，适合新建应用
- **SAML 2.0**：企业级兼容性最强，适合传统企业应用
- **OAuth2**：开放平台授权标准
- **CAS**：高校和旧版 SSO 系统
