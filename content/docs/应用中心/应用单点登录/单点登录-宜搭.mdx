## 常见问题与排障
### 单点登录失败的常见原因
1. sub值不匹配：
  - 检查集成平台中的属性值和声明名称配置
  - 确认宜搭中配置的用户标识字段与集成平台返回的sub值一致
2. Redirect URI不匹配：
  - 确保集成平台中配置的Redirect URI与宜搭中设置的完全一致
  - 注意检查URL中的大小写和特殊字符
1. 用户未关联：
  - 验证企业微信用户是否已与集成平台用户关联
  - 检查关联的账号状态是否正常
### 配置注意事项
## 域名使用：
  - Authorize URL和Token URL必须使用实际的租户域名，不是代理域名
  - 确保网络环境允许访问这些域名和端口
2. Public Key更新：
  - 如果遇到Token验证失败问题，尝试重新获取并更新Public Key
  - 注意移除jwks返回结果中的所有空格
1. 参数编码：
  - 确保URL中的特殊字符被正确编码
  - 特别注意redirect_uri参数的编码方式

### 术语解释
- OIDC：OpenID Connect，基于OAuth 2.0的身份认证协议
- jwks_uri：JSON Web Key Set的URI，用于获取验证Token签名的公钥
- sub：Subject的缩写，表示用户的唯一标识
- tcode：租户代码，用于识别特定的租户实例
