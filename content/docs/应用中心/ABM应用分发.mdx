## 概述

ABM 应用分发（Apple Business Manager）是专为苹果设备设计的企业应用集中分发方案。数犀集成平台与 ABM 集成，让企业能够统一管理 iOS/macOS 应用的购买、授权和分发，无需员工手动操作。

## 两种分发方式

| 方式 | 说明 | 适用场景 |
|------|------|---------|
| **VPP Code** | 通过批量采购计划（Volume Purchase Program）购买的应用授权码 | 一次性发放，员工自助兑换 |
| **ManagedID** | 通过 ABM 托管 Apple ID 进行授权分发 | 企业统一管控设备，推送安装 |

## 配置步骤

<Steps>
<Step>
### 上传 ABM 令牌

进入**应用中心 > ABM 应用分发 > 令牌管理**，上传从 Apple Business Manager 下载的令牌文件（`.p7m` 格式）。

**获取令牌方法：**
1. 登录 [Apple Business Manager](https://business.apple.com/)
2. 进入「偏好设置 > 付款和账单信息」
3. 下载令牌文件

| 参数 | 说明 |
|------|------|
| 令牌文件 | 从 ABM 下载的 `.p7m` 格式文件 |
| 组织信息 | 上传后自动显示企业名称和令牌过期时间 |

<Callout type="warn">
令牌有有效期，请在过期前及时更新，否则 ABM 同步功能将中断。
</Callout>
</Step>
<Step>
### 同步 ABM 应用

令牌上传成功后，系统自动同步 ABM 中已购买的应用列表。进入**ABM 应用分发**主页，可查看：
- 应用名称和图标
- 购买数量和已授权数量
- 应用详细信息
</Step>
<Step>
### 配置授权码（VPP Code 方式）

进入**ABM 应用分发 > 授权信息**，查看「VPP Code 兑换码状态」：

1. 点击「导入兑换码文件」，上传从 Apple VPP 获取的授权码文件
2. 系统显示已用数量和总数量
3. 点击「导出授权码」，将授权码分发给员工

</Step>
<Step>
### 向员工分发应用

根据员工设备情况选择分发方式：
- **VPP Code**：将授权码发给员工，员工在 App Store 兑换即可安装
- **ManagedID**：通过 MDM（移动设备管理）系统推送到托管设备，员工无需手动操作
</Step>
</Steps>

## 常见问题

<Accordions>
<Accordion title="同步后看不到 ABM 应用列表？">
1. 检查令牌是否已成功上传且未过期
2. 确认 ABM 账号中已有购买的应用
3. 尝试手动点击「刷新同步」重新拉取
</Accordion>
<Accordion title="令牌即将过期怎么办？">
在令牌过期前，前往 Apple Business Manager 下载新的令牌文件，回到「令牌管理」页面重新上传替换即可。不影响已分发的应用。
</Accordion>
<Accordion title="VPP Code 已全部用完怎么办？">
需要前往 Apple Business Manager 追加购买应用授权，购买完成后在平台导入新的授权码文件。
</Accordion>
</Accordions>
