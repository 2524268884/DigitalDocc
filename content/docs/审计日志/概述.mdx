## 审计日志体系

专属集成平台提供多维度的审计日志，帮助管理员追踪安全事件、排查问题和满足合规要求。

| 日志类型 | 记录内容 | 典型用途 |
|---------|---------|---------|
| [用户行为日志](./用户行为日志) | 用户登录、登出、密码修改等操作 | 安全审计、异常登录排查 |
| [用户变更日志](./用户变更日志) | 用户账号的增删改操作记录 | 追溯用户信息变更来源 |
| [管理员行为日志](./管理员行为日志) | 管理员在后台的操作记录 | 管理操作合规审计 |
| [接口调用日志](./接口调用日志) | 开放 API 和通讻录集成的调用记录 | 排查集成异常、监控调用频率 |
| [消息发送日志](./消息发送日志) | 钉钉/飞书消息通知发送记录 | 排查通知未收到问题 |

## 通用操作

所有日志页面均支持：
- **筛选**：按时间范围、操作类型、操作人等条件筛选
- **搜索**：关键词全文搜索
- **排序**：按时间倒序/正序排列
- **导出**：导出为 CSV/Excel 文件
