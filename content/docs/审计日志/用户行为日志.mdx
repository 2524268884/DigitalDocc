## 概述

用户行为日志记录员工对各业务应用的登录和访问行为，是安全审计和异常排查的核心工具。支持按时间、用户信息多维度筛选，并可导出至 Excel 供离线分析。

## 日志字段说明

| 字段 | 说明 |
|------|------|
| 操作时间 | 用户发起访问的具体时间 |
| 用户名 | 执行操作的用户账号 |
| 姓名 | 用户的真实姓名 |
| 应用名称 | 被访问的应用系统 |
| 操作类型 | 登录、登出等行为类型 |
| IP 地址 | 用户操作时的来源 IP |
| 操作结果 | 成功 / 失败 |

## 查询日志

<Steps>
<Step>
### 进入日志页面

在左侧导航栏选择**审计日志 > 用户行为日志**。
</Step>
<Step>
### 设置查询条件

支持以下筛选方式：

| 筛选维度 | 说明 |
|---------|------|
| 时间范围 | 默认显示最近一周，可自定义日期范围 |
| 姓名 | 支持模糊搜索 |
| 用户名 | 支持模糊搜索 |
| 手机号 | 精确匹配 |
| 邮箱 | 精确匹配 |

</Step>
<Step>
### 查看结果

点击「搜索」按钮，在列表中查看符合条件的行为记录。
</Step>
</Steps>

## 导出日志

点击页面右上角「导出」按钮，系统将当前筛选结果导出为 Excel 文件（`.xlsx`）。

<Callout type="info">
导出文件包含列表中所有可见字段，可直接对接外部日志分析系统使用。
</Callout>

## 常见问题

<Accordions>
<Accordion title="为什么查不到某个用户的日志？">
请检查：
- 时间范围是否覆盖目标时间段
- 用户是否在该时间段内有实际操作行为
- 搜索字段是否填写正确（注意区分用户名和姓名）
</Accordion>
<Accordion title="日志保存多长时间？">
系统默认保存最近 **180 天**的日志记录。如需调整，请在系统设置中查看或修改保存期限。
</Accordion>
</Accordions>
