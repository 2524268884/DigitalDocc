---

## 概述

管理员行为日志记录所有具有后台权限的管理员在数犀集成平台上执行的操作，包括用户管理、权限配置、系统设置等行为，是安全审计和合规管理的重要依据。

## 日志字段说明

| 字段 | 说明 |
|------|------|
| 操作时间 | 管理员执行操作的具体时间 |
| 操作者 | 执行操作的管理员账号 |
| IP 地址 | 操作时的来源 IP 地址 |
| 来源设备 | 操作时使用的设备信息 |
| 操作对象 | 被操作的系统资源或实体 |
| 操作类型 | 具体执行的操作类型 |
| 操作结果 | 成功 / 失败 |

## 查询日志

<Steps>
<Step>
### 进入日志页面

在左侧导航栏选择**审计日志 > 管理员行为日志**。
</Step>
<Step>
### 设置查询条件

| 筛选维度 | 说明 |
|---------|------|
| 时间范围 | 默认显示最近一周，可自定义日期范围 |
| 操作者 | 输入管理员用户名精确或模糊搜索 |

</Step>
<Step>
### 查看结果

点击「搜索」按钮，查看符合条件的管理员操作记录。
</Step>
</Steps>

## 导出日志

点击页面右上角「导出」按钮，将当前筛选结果导出为 Excel 文件（`.xlsx`）。

<Callout type="info">
导出文件包含页面上所有可见字段和符合筛选条件的全部记录，保留原始时间和 IP 信息，适合提交安全合规报告。
</Callout>

## 常见问题

<Accordions>
<Accordion title="为什么某个管理员的操作找不到记录？">
请检查：
- 时间范围是否正确（默认仅显示最近一周）
- 操作者字段输入是否为该管理员的**用户名**（非姓名）
- 该管理员是否在查询时间段内有实际后台操作
</Accordion>
<Accordion title="日志保存多长时间？">
系统默认保存最近 **180 天**的管理员操作日志。如需调整保存期限，请联系系统管理员在系统设置中修改。
</Accordion>
<Accordion title="如何快速定位某次误操作？">
建议结合「操作时间」和「操作者」双维度筛选，缩小范围后查看「操作对象」和「操作类型」字段确认具体操作内容。
</Accordion>
</Accordions>
