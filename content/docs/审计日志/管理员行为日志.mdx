## 概述

管理员行为日志用于记录系统管理员在专属集成平台上执行的所有操作，包括用户管理、权限设置、系统配置等。通过审计日志可以追踪管理员的操作行为，确保系统安全和合规性。

<Callout type="info">
日志默认保存最近 **180 天**，具体保存时间可在系统设置中查看或修改。
</Callout>

## 功能说明

### 查询条件

| 条件类型 | 说明 |
|---------|------|
| 时间范围 | 系统默认显示最近一周的操作日志记录，支持自定义选择开始日期和结束日期 |
| 操作者 | 支持通过管理员用户名进行搜索，快速定位特定管理员的操作记录 |

### 日志内容

管理员操作日志记录以下关键信息：

| 字段 | 说明 |
|-----|------|
| 操作时间 | 管理员执行操作的具体时间 |
| 操作者 | 执行操作的管理员账号 |
| IP 地址 | 管理员操作时的 IP 地址 |
| 来源设备 | 管理员操作时使用的设备信息 |
| 操作对象 | 被操作的系统资源或实体 |
| 操作结果 | 操作是否成功完成 |
| 操作类型 | 具体执行的操作类型 |

## 操作步骤

### 步骤 1：进入管理员行为日志页面

在左侧导航栏选择 **审计日志** → **管理员行为日志**。

### 步骤 2：设置查询条件

根据需要设置筛选条件：

- **时间范围**：通过日期选择器选择开始日期和结束日期（默认为最近一周）
- **操作者**：输入管理员用户名（可选）

### 步骤 3：执行搜索

点击「搜索」按钮执行查询。

### 步骤 4：查看日志列表

查看日志列表结果，包含操作时间、操作者、操作类型、操作对象、操作结果等字段。

### 步骤 5：导出日志（如需要）

点击右上角「导出」按钮，系统将生成 Excel 文件供下载，导出文件包含界面上显示的所有字段及符合筛选条件的全部记录。
