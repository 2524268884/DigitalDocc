## 概述

本指南详细说明如何在飞书开放平台创建应用、配置通讯录权限、获取应用凭证等操作步骤，帮助您快速完成飞书与系统的集成对接。通过正确配置，您可以实现组织架构同步、身份认证等核心功能。

## 前置条件

- 拥有飞书企业管理员账号
- 具备飞书开放平台应用开发权限
- 系统环境已准备就绪，可进行对接配置

## 应用创建与配置流程

<Steps>
<Step>
### 步骤 1：创建飞书应用

1. 访问[飞书开放平台](https://open.feishu.cn/app)并登录
2. 点击右上角「开发者后台」
3. 在开发者后台界面，点击「创建应用」按钮
4. 填写应用基本信息：
    - 应用名称
    - 应用图标
    - 应用描述
    - 应用分类
5. 确认信息后点击「创建」按钮

</Step>
<Step>
### 步骤 2：获取应用凭证

1. 应用创建完成后，在应用详情页面左侧菜单选择「凭证与基础信息」
2. 在页面中找到并记录以下关键参数：
    - AppID（应用唯一标识）
    - AppSecret（应用密钥）
> **重要提示**：AppSecret 为敏感信息，请妥善保管，避免泄露\[图片位置：飞书应用凭证获取界面截图\]
</Step>
<Step>
### 步骤 3：配置专属集成平台

1. 登录专属集成平台
2. 在左侧菜单导航至「参数配置 \> 授权管理」
3. 点击「添加授权」按钮
4. 在弹出的配置窗口中：
    - 自定义设置身份源名称（如"飞书身份源"）
    - 填写 AppID（即前面获取的 AppID）
    - 填写 AppSecret
    - 根据需要配置其他相关参数
5. 点击「确定」按钮保存配置

<Step>
### 步骤 4：配置应用权限

1. 返回飞书开放平台应用详情页
2. 在左侧菜单中选择「权限管理」
3. 添加以下必要权限：  **用户身份权限**：
    - 获取用户 ID
    - 获取用户基本信息
    - 获取用户邮箱
    - 获取用户手机号**通讯录权限**：
    - 获取通讯录组织架构信息
    - 获取通讯录单位信息
    - 获取通讯录部门基本信息
    - 获取通讯录部门组织架构信息
    - 获取通讯录用户基本信息
4. 重要：将通讯录权限范围设置为「全部成员」

<Step>
### 步骤 5：发布应用

1. 在应用详情页左侧菜单选择「应用发布 \> 版本管理与发布」
2. 点击「创建版本」按钮
3. 填写以下信息：
    - 应用版本号（如 1.0.0）
    - 版本更新说明
4. 设置可用性状态为「所有员工」
5. 点击「发布应用」按钮

<Step>
### 步骤 6：审核应用

1. 应用提交后，需进行企业管理员审核
2. 登录[飞书管理后台](https://www.feishu.cn/admin)
3. 导航至「工作台 \> 应用审核」
4. 找到待审核的应用，点击「审核」
5. 确认应用信息和权限后，点击「通过」按钮

## 参数说明

完成上述步骤后，您将获得以下关键参数用于系统集成：

参数名称说明获取位置AppID应用唯一标识应用详情「凭证与基础信息」AppSecret应用密钥，用于接口调用验证应用详情「凭证与基础信息」

## 验证配置

完成上述配置后，建议执行以下验证：
1. 在专属集成平台中执行连接测试
2. 验证通讯录数据同步功能
3. 测试身份认证流程

## 常见问题

<Accordions>
<Accordion title="权限配置失败">
**可能原因**：权限范围设置不正确或权限不足  **解决方案**：确保所有通讯录权限的范围均设置为「全部成员」
</Accordion>
<Accordion title="应用审核未通过">
**可能原因**：应用信息不完整或权限申请过多  **解决方案**：完善应用信息，仅申请必要的权限，详细说明权限用途
</Accordion>
<Accordion title="数据同步失败">
**可能原因**：AppID/AppSecret配置错误  **解决方案**：核对凭证信息，确保与飞书开放平台获取的一致
</Accordion>
</Accordions>
