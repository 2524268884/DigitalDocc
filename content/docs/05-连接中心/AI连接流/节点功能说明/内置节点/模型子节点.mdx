## 概述

**模型节点**是 AI Agent 的核心组件之一，负责处理自然语言输入、生成文本响应以及进行智能推理。合理配置模型节点，是构建智能流程的关键第一步。

### ✨ 模型节点的作用

模型节点的主要职责包括：
- 解析用户输入的自然语言内容
- 根据设定的提示词（Prompt）生成回答
- 支持上下文理解与意图识别
- 与内存节点、工具节点协同工作，实现智能多轮交互

通过模型节点，AI Agent 能够具备“语言理解 \+ 内容生成”的核心能力。

### 🧭 添加与配置模型节点
1. 插入模型节点

在 AI Agent 节点下方点击「模型插槽」

系统将弹出模型选择界面
2. 选择模型服务

支持主流语言大模型（LLM）平台，包括但不限于：

通义千问（阿里）

DeepSeek

OpenAI

Claude

选择模型时请确认：

已正确配置对应平台的 API Key / 密钥

模型版本满足业务需求（如语义理解、长文本处理）
3. 设置模型参数

可配置的核心参数包括：
- **温度（temperature）**：控制生成内容的随机性。值越大，生成内容越灵活多样，值越小则更稳定。示例值：0.7
- **频率惩罚系数（frequency penalty）**：对重复词语进行惩罚，提高语言多样性。示例值：0.5
- **存在惩罚系数（presence penalty）**：惩罚已出现过的词，鼓励使用新词，增加内容创新性。示例值：0.3
- **核采样概率（top-p）**：限制生成内容的概率分布范围，值越高表示内容越多样化。示例值：0.9
- **最大重试次数**：请求失败后的最大重试次数。用于提高请求成功率。示例值：3
- **最大生成 Token 数**：生成文本的最大长度限制，单位为 Token。示例值：1024
- **超时时间（秒）**：请求最大等待时长，超时将中止响应，避免卡死。示例值：60
