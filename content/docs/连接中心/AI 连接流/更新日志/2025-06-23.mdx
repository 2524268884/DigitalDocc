📢 AI 连接流功能更新说明（Release Note）
版本号：v1.0.0
发布日期：2025年6月23日
适用范围：连接流功能模块（集成平台 → 连接中心 → AI 连接流）
✨ 新功能
✅ 支持 AI Agent 节点能力配置
- 引入 AI Agent 节点，支持模型调用、上下文记忆、工具执行能力组合使用
- 可构建具备理解、记忆与行动能力的智能流程节点

✅ 新增子节点类型：模型、内存、工具
- 模型节点：可选择 GPT、通义千问、Claude、DeepSeek 等主流大模型，并支持参数精细化配置（温度、惩罚系数、上下文窗口等）
- 内存节点：支持短期/长期记忆配置，支持消息 key 跨流程传递上下文
- 工具节点：支持MCP Server 节点，实现 Agent 的外部能力调用
🧠 功能增强
🖼 画布全新升级
- AI 连接流已支持全新拖拽式画布，用户可通过拖拽节点快速构建流程，支持节点之间自由连接、移动与删除，整体交互更加直观高效

📝 流程日志能力升级
- AI 连接流现已支持在流程日志中查看 AI Agent 的完整运行链路，帮助用户更清晰地理解流程执行过程、定位异常原因。


🔮 即将支持
🧩 Chat Trigger 节点
- 支持通过可嵌入页面的对话组件触发流程执行
- 可识别当前用户身份信息，支持上下文透传与多轮交互
- 适用于业务系统集成、客服对话入口、嵌入式 AI 应用等场景
🐍 Python 脚本节点
- 支持流程中运行自定义 Python 脚本逻辑
- 可实现数据处理、接口调用、结果清洗等灵活操作
- 提供代码编辑器与执行结果展示
📚 知识库集成能力
- 支持上传企业文档构建知识库（PDF、DOCX、TXT 等）
- 新增知识库工具节点，结合模型检索与调用回答用户问题
- 支持文档分块、向量化存储、多轮追问等增强功能
