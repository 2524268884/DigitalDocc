## 概述

专属集成平台的短信服务默认使用平台内置短信签名。对于 SaaS 新开通的租户，若需将短信签名修改为企业自定义内容，可通过以下两种方式实现：

| 方式 | 适用场景 | 说明 |
|------|----------|------|
| **报备申请流程** | 希望使用自定义签名但不更换短信服务商 | 填写资料提交报备，由平台方完成签名更新 |
| **接入第三方短信服务** | 希望使用自有短信账户（如阿里云） | 在平台配置第三方短信网关，自主管理签名 |

---

## 方式一：报备申请流程

### 填写规范

提交报备前，请按照以下规范准备资料：

| 字段 | 要求 |
|------|------|
| **签名** | 不含空格及特殊符号（`./（）+~・?*&%#`），长度不超过 16 个字（不区分中英文） |
| **签名主体** | 不超过 64 个字 |
| **企业法人** | 法人姓名，可以是国外法人，但姓名不能有空格 |
| **单位证件类型** | 从下拉选择：1-营业执照；2-组织机构代码证；3-事业单位法人证书；4-社会团体法人登记证书；5-军队代码；6-个体户 |
| **企业信用代码** | 社会统一信用代码，不超过 18 位 |
| **企业责任人** | 负责该业务的人员姓名，只能是中国人 |
| **责任人身份证** | 16 位身份证号码，确保真实有效 |
| **固定扩展/用户扩展** | 根据实际情况填写，无特殊需求可留空 |

<Callout type="warn">
事业单位（如医院、学校、水电煤、交警、报社、各类协会等）除填写表格信息外，还需提供相关资质材料，且授权书最终要授权到「诚立业」，缺一不可。
</Callout>

### 报备步骤

<Steps>
<Step>
### 步骤 1：下载报备表格

下载「用户签名报备」表格模板（请联系技术支持获取），按照上方规范填写各项信息。
</Step>
<Step>
### 步骤 2：提交支持工单

提交技术支持工单，项目经理将拉群协助处理。
</Step>
<Step>
### 步骤 3：发送表格至指定群

将填写完整且无误的表格发送到指定工作群内进行报备。
</Step>
<Step>
### 步骤 4：等待审核生效

报备通过后，系统将更新短信签名配置，后续发送的短信将携带自定义签名。
</Step>
</Steps>

<Callout type="info">
请确保所填信息真实有效。责任人身份证号码及社会信用代码将用于签名资质审核，企业必须为国内正常注册的公司。
</Callout>

## 方式二：接入阿里云短信服务

若企业希望完全自主管理短信服务，可接入阿里云短信网关。

### 阿里云短信签名申请流程

<Steps>
<Step>
### 步骤 1：登录阿里云控制台

访问阿里云官方网站，使用企业账号登录，在控制台首页找到「短信服务」并进入。
</Step>
<Step>
### 步骤 2：申请短信签名

在短信服务控制台中，找到「签名管理」选项，点击「添加签名」，按页面提示填写以下信息：

- **签名内容**：符合阿里云规定，不含敏感词汇和特殊符号，一般不超过 16 字
- **签名类型**：根据实际情况选择企业签名、应用签名等
- **证明材料**：企业签名需上传营业执照照片，应用签名需提供应用资质文件
</Step>
<Step>
### 步骤 3：提交审核

确认填写信息和上传材料无误后，点击「提交审核」。

审核时间一般为 **1–3 个工作日**，审核结果通过短信或阿里云站内信通知。
</Step>
<Step>
### 步骤 4：在平台配置阿里云短信网关

签名审核通过后，参考「阿里云短信网关配置指引」在专属集成平台完成短信模板和模板 Code 等配置，即可使用自定义签名发送短信。
</Step>
</Steps>

## 常见问题

<Accordions>
<Accordion title="报备申请需要多久才能生效？">
具体时效取决于报备材料的完整性和审核进度，建议提前至少 5 个工作日准备材料提交。如有紧急需求，请在工单中注明优先级。
</Accordion>
<Accordion title="签名中可以包含英文或数字吗？">
可以，但签名中不能包含空格及特殊符号（`./（）+~・?*&%#`），且总长度（含英文和数字）不超过 16 个字符。
</Accordion>
<Accordion title="使用阿里云短信服务需要额外费用吗？">
是的，接入阿里云短信服务需使用企业自有的阿里云账户，短信发送费用由企业自行承担，与专属集成平台授权费用分开计算。
</Accordion>
<Accordion title="两种方式可以同时配置吗？">
不可以。平台只能使用一种短信发送渠道，接入第三方短信服务后，平台内置短信服务将不再使用。
</Accordion>
</Accordions>
