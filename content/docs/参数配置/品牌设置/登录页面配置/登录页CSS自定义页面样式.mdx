## 概述

专属集成平台支持通过自定义 CSS 对登录页面进行精细化样式定制。管理员可在内置代码编辑器中编写 CSS，左侧预览区域会实时响应修改结果，满足企业品牌化定制需求。

**配置路径**：品牌设置 → 登录页面设置 → 登录页设置 → 开启「自定义页面样式」

---

## 操作步骤

<Steps>
<Step>
### 步骤 1：进入登录页设置

导航至 **品牌设置 → 登录页面设置**，找到「登录页设置」表单。
</Step>
<Step>
### 步骤 2：开启自定义页面样式

在表单内容中，打开「自定义页面样式」开关，页面将显示代码编辑器。
</Step>
<Step>
### 步骤 3：输入 CSS 样式

在代码编辑器中输入自定义 CSS 样式，左侧预览区域会实时展示修改效果。

例如，修改标题字体颜色：

```css
.digitalsee-login-header-lead {
  color: blue;
}
```
</Step>
<Step>
### 步骤 4：保存配置

确认预览效果符合预期后，点击「保存」按钮使配置生效。
</Step>
</Steps>

---

## 可自定义的样式区域

以下为目前开放的可修改样式类名及对应区域：

| 样式类名 | 对应区域 |
|----------|----------|
| `.digitalsee-box-container` | 整体容器 |
| `.digitalsee-login-header` | 头部容器 |
| `.digitalsee-login-header-logo` | Logo 区域 |
| `.digitalsee-login-header-title` | 标题容器 |
| `.digitalsee-login-header-lead` | 标题文字 |
| `.digitalsee-policy-text-container` | 隐私策略区域 |
| `.digitalsee-login-button` | 登录按钮 |
| `.digitalsee-forget` | 忘记密码链接 |
| `.digitalsee-remember` | 记住密码选项 |
| `.digitalsee-other-login-methods-box` | 其他登录方式容器 |

---

## 示例代码

### 修改标题颜色

```css
.digitalsee-login-header-lead {
  color: #1a73e8;
  font-size: 24px;
  font-weight: bold;
}
```

### 自定义登录按钮样式

```css
.digitalsee-login-button {
  background-color: #0052cc;
  border-radius: 8px;
  font-size: 16px;
}
```

### 调整容器背景和圆角

```css
.digitalsee-box-container {
  background-color: rgba(255, 255, 255, 0.9);
  border-radius: 12px;
  box-shadow: 0 4px 20px rgba(0, 0, 0, 0.15);
}
```

<Callout type="warn">
自定义 CSS 仅支持对上方列出的样式类进行修改，请勿使用 `!important` 覆盖平台核心样式，以免影响登录功能正常运行。
</Callout>

<Callout type="info">
建议修改前备份原有 CSS 代码，以便在需要时快速恢复。
</Callout>

## 常见问题

<Accordions>
<Accordion title="CSS 修改后预览正常但保存后不生效怎么办？">
请确认已点击「保存」按钮，并尝试清除浏览器缓存后刷新页面（Ctrl+F5 或 Cmd+Shift+R）。
</Accordion>
<Accordion title="能否修改登录表单中的输入框样式？">
目前开放的样式类不包含输入框，仅支持上方列出的区域。如有更多定制需求，请联系技术支持。
</Accordion>
<Accordion title="自定义 CSS 是否会影响移动端登录页显示？">
CSS 会对所有设备端的登录页生效，建议使用响应式 CSS 或 media query 对移动端做适配调整。
</Accordion>
</Accordions>
