---
title: "登录页CSS自定义页面样式"
---

如想对登录页面，进行CSS自定义样式修改，可在“品牌设置-登录页面设置”，“登录页设置”的表单内容中，开启“自定义页面样式”。在代码编辑器中，输入CSS样式，进行修改。




例如想要修改标题字体颜色，可在代码编辑器中输入

```css
.digitalsee-login-header-lead {
  color: blue;
}
```

左侧的预览效果，即可实时响应修改结果



目前开放的，可修改的页面样式如下
区域样式名
	
对应修改区域


/* 容器 */
```css
.digitalsee-box-container {
}
```





/* 头部容器 */
```css
.digitalsee-login-header {
}
```





/* Logo */
```css
.digitalsee-login-header-logo {
}
```
/* 标题容器 */
```css
.digitalsee-login-header-title {
}
```
/* 标题 */
```css
.digitalsee-login-header-lead {
}
```
/* 隐私策略 */

}

}
