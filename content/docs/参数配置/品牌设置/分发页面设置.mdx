## 概述

分发页面设置允许管理员自定义应用分发平台的界面样式、内容和验证方式，为企业用户提供品牌化的应用下载体验。

**配置路径**：品牌设置 → 分发页面设置

---

## 功能说明

### 页面定制配置项

| 配置项 | 功能说明 | 格式/限制 |
|--------|----------|-----------|
| **页面 Logo** | 设置分发页面的企业标识 | PNG/JPG，不超过 500 KB，支持在线裁剪 |
| **页面背景图** | 设置分发页面的背景图像 | PNG/JPG，不超过 500 KB，不上传时显示白色背景 |
| **按钮颜色** | 自定义页面按钮颜色 | 十六进制颜色代码，默认紫色 `#7936C3` |
| **标题文案** | 设置分发页面的主标题（必填） | 支持中英文，用 `|` 分隔，如 `欢迎使用\|Welcome` |
| **验证引导文案** | 设置验证页面的提示文字 | 支持中英文，用 `|` 分隔 |
| **底部提示文案** | 设置页面底部说明文字 | 支持中英文，用 `|` 分隔 |
| **下载说明** | 提供应用下载的详细说明和指引 | 富文本编辑器，支持中英文双语 |

<Callout type="info">
多语言内容使用半角竖线 `|` 分隔中英文，例如：`欢迎使用应用分发平台|Welcome Download`。
</Callout>

### 验证方式

控制用户访问分发页面时的验证需求：

| 验证方式 | 说明 |
|----------|------|
| **不验证**（默认） | 用户可直接进入下载页面 |
| **提取码** | 需要输入提取码才能进入下载页面 |
| **短信验证** | 需要通过短信验证码验证才能进入下载页面 |

<Callout type="warn">
验证方式的变更会直接影响用户下载应用的流程，请在变更前通知相关用户，并谨慎操作。
</Callout>

### 下载地址

系统会为每个租户自动生成两种唯一的下载链接：

- **VPP Code 下载地址**：用于通过 Apple VPP 计划分发的应用
  ```
  https://poc.xxxxxxx.cn/adm/download/?tenantId=poc&channel=VPP_CODE
  ```

- **Managed ID 下载地址**：用于通过 Managed Apple ID 分发的应用
  ```
  https://poc.xxxxxxx.cn/adm/download/?tenantId=poc&channel=MANAGEID
  ```

## 最佳实践

- **Logo 设计**：建议使用透明背景的 PNG 格式企业 Logo，保持边缘清晰
- **背景图选择**：选择低饱和度、不干扰文字阅读的背景图片
- **按钮颜色**：与企业 VI 系统保持一致，确保与背景色形成足够对比
- **多语言内容**：保证中英文内容意义一致，英文部分注意大小写和标点规范
- **下载说明**：内容简洁明了，突出重点信息，适当使用标题和列表增强可读性
- **预览验证**：建议在正式发布前，通过分发链接预览效果，确保显示正常

<Callout type="info">
页面定制设置保存后即时生效，会影响所有用户的分发页面体验。分发页面设置仅影响前端页面，不影响应用分发的后端功能和流程。
</Callout>

## 常见问题

<Accordions>
<Accordion title="上传 Logo 后不显示怎么办？">
请检查图片大小是否超过 500 KB，以及格式是否为 PNG 或 JPG。如仍有问题，可尝试删除后重新上传并保存。
</Accordion>
<Accordion title="如何恢复默认 Logo 或背景图？">
直接删除已上传的图片并保存即可，系统将自动恢复显示默认值。
</Accordion>
<Accordion title="短信验证方式需要额外配置吗？">
是的，使用短信验证方式前需先在「品牌设置 → 短信模板配置」中完成短信服务的配置。
</Accordion>
</Accordions>
