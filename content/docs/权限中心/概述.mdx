## 什么是权限中心

权限中心负责管理平台内所有的访问控制策略，包括管理员权限体系和用户对应用的授权关系。

## 核心功能

| 功能 | 说明 |
|------|------|
| [管理员账号](./管理员账号) | 管理具有后台管理权限的账号 |
| [管理员角色](./管理员角色) | 定义不同级别的管理员职责范围 |
| [基于组织架构授权](./用户授权管理/基于组织架构和用户授权) | 按部门/用户分配应用访问权限 |
| [基于角色授权（RBAC）](./用户授权管理/基于角色授权（RBAC）) | 通过角色实现批量权限管理 |
| [用户自助申请](./用户授权管理/用户自主申请应用（OA审批授权）权限) | 用户发起申请，走 OA 审批流 |

## 权限层级说明

数犀集成平台采用两层权限模型：

```
超级管理员
  └── 管理员角色（自定义权限范围）
        └── 普通管理员账号

用户
  └── 应用权限（直接授权 / 角色授权 / 申请授权）
```

**管理员权限**：控制谁能管理平台配置
**用户应用权限**：控制哪些用户能访问哪些应用
