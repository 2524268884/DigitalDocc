## 概述

用户（尤其是需要临时访问特定应用的员工）可以在自助门户主动申请应用权限，无需联系管理员或填写线下申请单。申请提交后，管理员在钉钉工作台收到审批通知，通过后权限自动生效，全程留档可追溯。

## 申请流程

```
用户进入工作台 → 点击应用「申请」按钮 → 填写申请原因和使用时间 → 提交申请
       ↓
管理员收到钉钉审批通知 → 点击「同意」或「拒绝」
       ↓
审批通过 → 权限自动生效，用户可访问应用
```

## 用户申请步骤

<Steps>
<Step>
### 进入自助门户

点击管理平台右上角的「用户服务台」图标，进入自助门户工作台页面。
</Step>
<Step>
### 找到目标应用并申请

在工作台的「应用中心」中，找到需要申请访问权限的应用（无权限的应用右上角会显示「申请」标志），点击「申请」按钮。
</Step>
<Step>
### 填写申请信息

在弹出的申请表单中填写：

| 参数 | 说明 | 是否必填 |
|------|------|---------|
| 申请原因 | 说明业务需求和使用目的 | 是 |
| 使用时间 | 预计使用的时间段 | 是 |
| 权限范围 | 申请的具体权限（如应用支持）| 否 |

点击「提交」完成申请。
</Step>
</Steps>

## 管理员审批步骤

<Steps>
<Step>
### 接收审批通知

申请提交后，管理员的钉钉工作台会收到审批通知消息。
</Step>
<Step>
### 处理审批

在钉钉审批通知中，查看申请详情后点击「同意」或「拒绝」。

<Callout type="info">
审批通过后，系统自动为申请用户授予该应用的访问权限，无需管理员手动操作。
</Callout>
</Step>
</Steps>

## 账号续期申请

除应用权限申请外，用户还可以在自助门户申请账号续期：

1. 进入自助门户，点击右上角进入「个人中心」
2. 点击「账号续期申请」，填写续期申请信息提交
3. 同样通过钉钉审批流完成，审批结果推送给管理员

## 常见问题

<Accordions>
<Accordion title="申请提交后多久会有审批结果？">
审批时效由管理员决定。提交后管理员的钉钉工作台会立即收到通知，通常建议 1 个工作日内完成审批。如长时间未得到响应，可联系管理员催办。
</Accordion>
<Accordion title="申请被拒绝后可以重新申请吗？">
可以。申请被拒绝后，可以在自助门户重新提交申请，建议在申请原因中补充更详细的业务说明，提高审批通过率。
</Accordion>
<Accordion title="临时权限到期后会自动收回吗？">
根据配置，设置了「使用时间」的权限到期后系统会自动收回。管理员也可以在授权管理页面手动提前收回权限。
</Accordion>
</Accordions>
