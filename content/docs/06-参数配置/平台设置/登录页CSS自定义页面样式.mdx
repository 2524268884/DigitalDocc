## 概述

如想对登录页面，进行CSS自定义样式修改，可在“品牌设置-登录页面设置”，“登录页设置”的表单内容中，开启“自定义页面样式”。在代码编辑器中，输入CSS样式，进行修改。

例如想要修改标题字体颜色，可在代码编辑器中输入
```text
.digitalsee-login-header-lead {
  color: blue;
}
```
