数犀集成平台是面向企业的 **IDaaS + iPaaS 融合平台**，专注于统一身份管理与系统集成，帮助企业打通账号、通讻录、应用与业务系统，构建无缝连接的数字化基础设施。

## 核心能力

| 能力模块 | 说明 |
|---------|------|
| **通讻录集成** | 从钉钉、飞书、企微、AD/LDAP 等上游系统拉取组织架构和用户数据 |
| **通讻录同步** | 将平台用户数据推送至 OA、HR、ERP 等下游业务系统 |
| **单点登录（SSO）** | 支持 OIDC / SAML / OAuth2 / CAS 协议，一次登录全平台通行 |
| **认证源管理** | 对接钉钉、飞书、企微、AD、LDAP、微信等多种身份认证方式 |
| **权限管理** | 基于角色（RBAC）和组织架构的应用访问控制 |
| **AI 连接流** | 可视化 AI 工作流编排，集成大模型与业务系统 |
| **连接流** | 自动化数据流转，连接各类 SaaS 和业务系统 |

## 典型使用场景

- **统一登录门户**：员工使用钉钉/飞书账号一键登录所有企业应用
- **新员工入职**：HR 系统创建账号，自动同步到钉钉/OA/邮件系统
- **离职账号注销**：上游组织架构变更，自动停用下游系统账号
- **多系统数据打通**：ERP、CRM、HR 系统间数据自动流转
- **AI 业务自动化**：结合大模型处理审批、客服、报表等智能场景

## 产品优势

- **双向数据流**：支持上游拉取（集成）和下游推送（同步）两个方向
- **协议全覆盖**：SSO 支持 OIDC / SAML / OAuth2 / CAS，认证源覆盖主流身份系统
- **开箱即用**：内置钉钉、飞书、企微等常用连接器，无需重复开发
- **部署灵活**：支持公有云 SaaS 和私有化部署

<Callout type="info">
数犀集成平台已在金融、制造、医疗、教育等多个行业成功落地，支持私有化部署和公有云两种模式。
</Callout>
