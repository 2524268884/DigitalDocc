## 概述

在将钉钉接入数犀集成平台之前，需要先在钉钉开放平台创建一个企业内部应用，获取接口调用凭证，并配置必要的权限和回调地址。

## 前提条件

| # | 条件 | 验证方式 |
|---|------|---------|
| 1 | 钉钉企业管理员账号 | 能登录[钉钉开放平台](https://open.dingtalk.com/) |
| 2 | 数犀集成平台管理员账号 | 能登录集成平台管理控制台 |

---

## 钉钉侧配置

<Steps>
<Step>
### 获取企业 CorpId

1. 登录[钉钉开放平台](https://open.dingtalk.com/)
2. 在首页或「企业信息」页面查看并记录 **企业 CorpId**

<Callout type="info">
CorpId 是企业唯一标识，后续在集成平台配置授权时需要填写。
</Callout>
</Step>
<Step>
### 创建企业内部应用

1. 在顶部导航选择**应用开发 > 企业内部开发**
2. 点击「钉钉应用」，选择**小程序**（H5 微应用）
3. 点击「创建应用」，填写以下信息：

| 字段 | 说明 |
|------|------|
| 应用名称 | 填写便于识别的名称，如「数犀集成平台」 |
| 应用图标 | 上传应用图标 |
| 应用简介 | 简要描述应用用途 |

4. 确认后点击「创建」
![创建企业内部应用-1](/images/快速开始/钉钉开放平台创建应用/image-001.png)
![创建企业内部应用-2](/images/快速开始/钉钉开放平台创建应用/image-002.png)
</Step>
<Step>
### 配置服务器出口 IP

1. 在应用管理页左侧菜单点击**开发管理**
2. 找到「服务器出口 IP」配置项
3. 填入数犀集成平台的服务器出口 IP 地址

<Callout type="warn">
必须正确配置出口 IP，否则集成平台无法调用钉钉 API。私有化部署请填写您自己的服务器出口 IP。
</Callout>
</Step>
<Step>
### 获取 AppKey 和 AppSecret

1. 在应用管理页左侧菜单点击**基础信息**
2. 记录以下关键参数：

| 参数 | 说明 | 用途 |
|------|------|------|
| `AppKey` | 应用唯一标识 | 填入集成平台授权配置 |
| `AppSecret` | 应用密钥 | 填入集成平台授权配置 |

<Callout type="warn">
AppSecret 是敏感信息，请妥善保管，不要泄露给无关人员。
</Callout>
![获取AppKey-1](/images/快速开始/钉钉开放平台创建应用/image-003.png)
![获取AppKey-2](/images/快速开始/钉钉开放平台创建应用/image-004.png)
</Step>
<Step>
### 开通接口权限

1. 在左侧菜单点击**权限管理 > 接口权限**
2. 添加以下必要权限：

| 权限名称 | 用途 |
|---------|------|
| 通讻录身份验证 | 用户身份认证 |
| 成员信息读取 | 同步用户信息 |
| 部门信息读取 | 同步组织架构 |
| 企业信息读取 | 获取企业基本信息 |

<Callout type="info">
如需使用**专属钉钉**的文件审计、SSO 账号禁用/启用功能，还需额外开通「访问控制」和「账号管理」权限。
</Callout>
![开通接口权限-1](/images/快速开始/钉钉开放平台创建应用/image-005.png)
![开通接口权限-2](/images/快速开始/钉钉开放平台创建应用/image-006.png)
![开通接口权限-3](/images/快速开始/钉钉开放平台创建应用/image-007.png)
</Step>
<Step>
### 配置回调域名

1. 在左侧菜单点击**应用功能 > 登录与分享**
2. 点击回调地址的「添加」按钮
3. 填入数犀集成平台的租户域名，例如：

```
https://your-tenant.idaas-domain.com
```
</Step>
</Steps>

## 集成平台侧配置

<Steps>
<Step>
### 添加钉钉授权

1. 登录数犀集成平台管理控制台
2. 进入**系统设置 > 授权管理**
3. 点击「添加授权」，选择「钉钉」
</Step>
<Step>
### 填写凭证信息

在配置表单中填写以下参数：

| 配置项 | 说明 | 值 |
|--------|------|-----|
| 授权名称 | 自定义名称，便于识别 | 如「钉钉生产环境」 |
| 企业 ID（CorpId）| 前面记录的企业 CorpId | — |
| AppKey | 钉钉应用的 AppKey | — |
| AppSecret | 钉钉应用的 AppSecret | — |

3. 点击「确定」保存
![填写凭证信息-1](/images/快速开始/钉钉开放平台创建应用/image-008.png)
![填写凭证信息-2](/images/快速开始/钉钉开放平台创建应用/image-009.png)
![填写凭证信息-3](/images/快速开始/钉钉开放平台创建应用/image-010.png)
</Step>
</Steps>

---

## 验证配置

配置完成后，点击授权记录旁的「测试连接」，确认返回成功状态。

如连接失败，请检查：
- CorpId / AppKey / AppSecret 是否正确复制（注意不含多余空格）
- 服务器出口 IP 是否已正确配置
- 接口权限是否已全部开通

## 常见问题

<Accordions>
<Accordion title="为什么调用钉钉 API 返回「IP 未在白名单」错误？">
请检查**服务器出口 IP** 是否已正确添加到钉钉应用配置中。公有云部署请联系数犀集成平台客服获取出口 IP；私有化部署请填写您自己服务器的出口 IP。
</Accordion>
<Accordion title="AppSecret 忘记了怎么办？">
前往钉钉开放平台 → 应用详情 → 基础信息，可以重置 AppSecret。**重置后旧的 AppSecret 立即失效**，需同步更新集成平台授权配置。
</Accordion>
<Accordion title="专属钉钉和普通钉钉配置有什么区别？">
专属钉钉需要额外开通「访问控制」和「账号管理」两个接口权限，才能使用 SSO 账号禁用/启用等高级功能。其余配置步骤完全相同。
</Accordion>
</Accordions>
