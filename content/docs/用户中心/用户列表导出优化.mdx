---

## 功能概述

专属集成平台的用户列表导出功能已进行优化，管理员在导出数据时可以自主选择导出范围和导出字段，解决之前导出数据范围不明确的问题。

**优化内容：**

| 优化项 | 优化前 | 优化后 |
|-------|-------|-------|
| 导出范围 | 不明确，管理员无法区分导出的是当前页还是全部 | 支持选择「当前页」或「全部数据」 |
| 导出字段 | 固定字段，与用户当前配置无关 | 支持选择「全部字段」或「仅显示字段」 |
| 筛选后导出 | 无法区分导出的是筛选前还是筛选后的数据 | 支持选择「全部数据」或「筛选后数据」 |

## 使用步骤

<Steps>
<Step>
### 步骤一：进入用户列表

在专属集成平台管理控制台中，进入 **用户中心 > 用户管理**，查看用户列表。
</Step>
<Step>
### 步骤二：配置筛选条件（可选）

如需导出特定范围的用户，可先在上方筛选栏中配置筛选条件（如部门、状态、入职时间等）。
</Step>
<Step>
### 步骤三：点击导出按钮

点击列表右上角的「导出」按钮，弹出导出配置弹窗。
</Step>
<Step>
### 步骤四：配置导出选项

在导出配置弹窗中，选择以下选项：

| 配置项 | 选项 | 说明 |
|--------|------|------|
| 导出范围 | 当前页 | 仅导出当前页面展示的用户（约 20 条） |
| | 全部数据 | 导出当前筛选条件下所有用户 |
| 导出字段 | 全部字段 | 导出用户所有字段（包括隐藏列） |
| | 仅显示字段 | 仅导出用户在列表中配置为显示状态的字段 |
| 数据范围 | 全部数据 | 忽略当前筛选条件，导出全部用户 |
| | 筛选后数据 | 仅导出经过筛选后的用户 |

<ImageZoom src="/images/user/user-export-options.png" alt="用户导出配置界面" width={800} height={450} />
</Step>
<Step>
### 步骤五：确认导出

点击「确认导出」，系统开始生成 Excel 文件并在完成后自动下载。

<Callout type="info">
导出全部数据时，如数据量超过 10,000 条，系统会分批生成并压缩为 ZIP 文件下载。
</Callout>
</Step>
</Steps>

## 常见问题

<Accordions>
<Accordion title="导出「仅显示字段」与用户在列表中的列配置是实时同步的吗?">
是的。「仅显示字段」会读取用户在列表页的列配置状态，管理员可在列表页右上角的「列配置」中调整哪些字段显示或隐藏。
</Accordion>
<Accordion title="导出文件中的字段顺序是什么?">
导出字段顺序与「仅显示字段」模式下的列顺序保持一致，未显示的字段不会出现在导出文件中。
</Accordion>
</Accordions>
